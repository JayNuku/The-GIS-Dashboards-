# GIS Dashboard Project

This repository contains a simple GIS dashboard prototype and an hourly production report template.

## Files

- `gis-dashboard.html` — interactive dashboard prototype with metrics, map-style visualization, layer table, and charts.
- `work.html` — printable hourly production report layout.
- `css/styles.css` — extracted dashboard styling for cleaner structure.
- `js/dashboard.js` — extracted dashboard interaction and chart scripts.

## How to use

1. Open `gis-dashboard.html` in a browser to see the dashboard.
2. Open `work.html` in a browser to see the production report form.

## Notes

- The dashboard uses Chart.js from the CDN.
- The dashboard is currently a static mockup; next steps are to add real GIS data, map integration, and live data-driven charts.
